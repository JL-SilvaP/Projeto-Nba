🏀 NBA Manager System

Sistema desenvolvido em Java para gerenciamento de jogadores, técnicos e times da NBA, com foco em organização de dados e análise de estatísticas.

📌 Descrição

O NBA Manager System é um sistema orientado a objetos que permite cadastrar e gerenciar informações relacionadas à NBA, como jogadores, técnicos e times.

Ele foi desenvolvido com o objetivo de facilitar a organização e visualização de dados esportivos de forma simples e eficiente.

🎯 Objetivo

Criar um sistema em Java que possibilite:

Cadastro de jogadores, técnicos e times
Organização de dados em estruturas orientadas a objetos
Armazenamento e visualização de estatísticas dos jogadores
⚙️ Funcionalidades
✅ Cadastro de jogadores
✅ Cadastro de técnicos
✅ Criação e gerenciamento de times
✅ Registro de estatísticas dos jogadores
✅ Listagem de jogadores, técnicos e times
✅ Gerenciamento central do sistema
🏗️ Estrutura do Projeto

O projeto é composto pelas seguintes classes:

src/
│
├── App.java                  # Classe principal (execução)
├── Pessoa.java              # Classe base
├── Jogador.java             # Representa jogadores
├── Tecnico.java             # Representa técnicos
├── Time.java                # Representa times
├── EstatisticasJogador.java # Estatísticas dos jogadores
├── Governador.java          # Administração do sistema
├── ManagerGeral.java        # Controle geral do sistema

🧠 Conceitos Utilizados
Programação Orientada a Objetos (POO)
Encapsulamento
Herança
Abstração
Organização em classes
Manipulação de dados
